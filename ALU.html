ALU Architecture

An ALU takes two n-bit inputs (operand A and operand B), an operation selector (opcode), and a carry-in bit. It outputs a result register and a set of condition flags that reflect the outcome of the operation.

// INPUTS
A [7:0] — Operand A (8 bits)
B [7:0] — Operand B (8 bits)
OP [3:0] — Operation selector
Cin [0] — Carry in
// OUTPUTS
Result [7:0] — Computed result
Z [0] — Zero flag
C [0] — Carry flag
V [0] — Overflow flag
N [0] — Negative flag

1-Bit Full Adder

A full adder computes the sum of three 1-bit inputs: A, B, and a Carry-in (Cin). It outputs a Sum bit and a Carry-out (Cout). Chain 8 of these to build an 8-bit adder.

// INPUTS
A Bit A
B Bit B
Cin Carry in
SUM
0
CARRY OUT
0
// BOOLEAN EXPRESSIONS
Sum = A ⊕ B ⊕ Cin
Cout = (A·B) + (B·Cin) + (A·Cin)
// CURRENT STATE
A=0, B=0, Cin=0 → Sum=0, Cout=0
